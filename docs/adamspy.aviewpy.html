adamspy.aviewpy.aviewpy.no_start_scripts(cwd: pathlib.Path, skip: List[str] = None, preserve_log_file=True)[source]¶

Moves all startup script files NOT named in :arg:skip to a temporary directory on entry. Moves them back on exit.

Parameters:
  • cwd (Path) – Current working directory
  • scripts (str, optional) – Names of scripts to move. Nonexistent files are ignored, by default [‘aviewBS.cmd’, ‘aview.cmd’, ‘aviewAS.cmd’]
